Implementation Summary

CINeMA-Browser was implemented as a single HTML file (1,670 lines) with no external dependencies. The tool runs entirely in the browser with no network requests after initial load, providing true offline capability. The implementation covers all six CINeMA assessment domains as specified in Papakonstantinou et al. (2020).

A Canvas-based force-directed graph algorithm renders the treatment network in real time, with nodes representing treatments and edges representing direct comparisons. Edge width encodes the number of contributing studies, providing an immediate visual summary of the evidence base.

Export Formats

  • HTML report: Full styled assessment table with domain justifications, ready for inclusion in supplementary materials
  • CSV summary: Machine-readable domain ratings and narratives for downstream analysis
  • Narrative paragraph: Plain-language summary suitable for manuscript Methods/Results sections
  • Clipboard copy: One-click copy of the assessment table for pasting into word processors

Availability

The tool is freely available with no registration requirement:

  • Live URL: https://mahmood726-cyber.github.io/cinema/cinema.html
  • Source code: https://github.com/mahmood726-cyber/cinema
  • License: MIT
This tool requires only a modern web browser and works fully offline after the initial page load. No installation, registration, or internet connection is required for use.

Limitations

  • Auto-suggestions are rule-based and intended as guidance only; final domain ratings require expert judgment
  • The tool does not import NMA results directly from statistical software; numeric inputs must be entered manually
  • localStorage capacity limits apply for very large assessments (>5 MB in most browsers)

Conclusion

CINeMA-Browser provides the first free, offline-capable, single-file browser implementation of the CINeMA framework for confidence assessment in network meta-analysis. The tool correctly implements all six domains, provides evidence-informed auto-suggestions, renders interactive network graphs, and exports results in four formats. It is immediately available for use by systematic reviewers, methodologists, and guideline developers worldwide.
