CINeMA-Browser: A Free Browser-Based Tool for Confidence in Network Meta-Analysis — Protocol

Mahmood Alhusseini
London Cardiology Clinic, London, United Kingdom
Status: Complete Registered: 2026-03-31 Version: 1.0 Format: E156 Protocol

Background

Network meta-analysis (NMA) synthesises evidence from multiple treatments compared across a network of studies, enabling indirect and mixed comparisons not available in any single trial. The CINeMA (Confidence in Network Meta-Analysis) framework, developed by Papakonstantinou et al. (2020), provides a structured approach for grading the confidence in NMA evidence across six methodological domains.

Despite widespread adoption of the CINeMA framework in systematic reviews and health technology assessments, no freely available offline browser tool exists that implements the full assessment workflow with auto-suggestions, network visualisation, and publication-ready output. Existing implementations require institutional access or online connectivity, limiting use in low-resource settings and creating reproducibility gaps when online services change.

Objectives

Primary: Build a single-file, offline-capable browser tool that fully implements the CINeMA framework for rating confidence in NMA estimates across all six assessment domains.

Secondary:

  • Provide evidence-informed auto-suggestions for each domain rating to guide users
  • Render an interactive force-directed network graph showing treatment connections and evidence flow
  • Generate publication-ready traffic-light summary tables in multiple export formats
  • Enable persistent project storage via localStorage for multi-session workflows
  • Embed three worked clinical examples for immediate use and user orientation

Methods

Component Implementation
Architecture Single HTML file with embedded CSS and JavaScript — no build step, no server, no dependencies
Network Visualisation Canvas-based force-directed graph; nodes = treatments, edges = direct comparisons; edge width encodes study count
CINeMA Domains Within-study bias, Reporting bias, Indirectness, Imprecision, Heterogeneity, Incoherence — each rated High/Moderate/Low confidence with narrative justification
Auto-suggestions Rule-based suggestions triggered by entered study counts, weights, I² values, and comparison type (direct/indirect/mixed)
Storage localStorage key cinema_v1_projects; schema version 1; migration logic included
Export HTML report, CSV summary, narrative paragraph, clipboard copy
Examples 3 built-in worked examples covering antidepressants, antihypertensives, and antiplatelets networks
Safety checks Div balance, script integrity, function uniqueness, ID uniqueness, XSS escaping verified

Outcomes

Primary outcome: Correct CINeMA domain assessment with traffic-light confidence table matching the published CINeMA methodology (Papakonstantinou 2020), verified against the three built-in clinical examples.

Secondary outcomes:

  • Export of completed assessments as HTML, CSV, and narrative text
  • Offline functionality (no network requests after initial load)
  • Sub-second load time on standard hardware
  • Accessible keyboard navigation and ARIA labelling for all interactive elements
